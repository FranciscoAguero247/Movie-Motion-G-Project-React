- **Dynamic Search-Priority UI:** A custom-engineered navigation bar that prioritizes the search experience on mobile devices while maintaining a clean desktop layout.
- **Smart Recommendations:** Automatically suggests similar movies based on the genre of the selected title.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using CSS Grid and Flexbox.
